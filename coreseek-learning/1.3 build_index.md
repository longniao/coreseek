# Coreseek 建立索引

### 1. 建立索引
```
# 建立全部索引
/home/service/coreseek/bin/indexer -c /home/service/coreseek/etc/sphinx_search.conf --all

# 建立增量索引
/home/service/coreseek/bin/indexer -c /home/service/coreseek/etc/sphinx_search.conf search_delta_9312

# 启动搜索服务（监听端口）
/home/service/coreseek/bin/searchd -c /home/service/coreseek/etc/sphinx_search.conf

# 搜索服务启动后重新全量索引
/home/service/coreseek/bin/indexer -c /home/service/coreseek/etc/sphinx_search.conf --all --rotate

# 命令行查询
/home/service/coreseek/bin/search -c /home/service/coreseek/etc/sphinx_search.conf 花千骨

# 关闭搜索服务	
/home/service/coreseek/bin/searchd -c /home/service/coreseek/etc/sphinx_search.conf --stop

```
### 2. 合并增量索引
```
/home/service/coreseek/bin/indexer -c /home/service/coreseek/etc/sphinx_search.conf --merge search_9312 search_delta_9312 --rotate --merge-dst-range deleted 0 0
```

### 3. 参数解释
```
--rotate 在搜索服务启动后重建索引需要加次选项，未启动不需要加

--merge-dst-range deleted 0 0   为了防止多个关键字指向同一个文档加上
这个开关允许您在合并过程中对目标索引实施过，过滤器可以有多个，只有满足全部过滤条件的文档才会在最终合并后的索引中出现。

--merge-killists 去旧（强行更新，3.2版本好像有bug） 
用法如下：
/home/service/coreseek/bin/indexer indexer --merge search_9312 search_delta_9312 --rotate --merge-killists

```
